GhostDriver

Ghost Driver is an implementation of the Remote WebDriver Wire protocol, using PhantomJS as back-end

Download as .zip Download as .tar.gz View on GitHub

Forewords

As for now, GhostDriver is a technical-demo. I worked on this 1 week and the idea is to present it to the Selenium Conference 2012. It's not usable as a normal WebDriver yet, but it will be in the foreseeable future.

Important, please read

  1. IT'S NOT DONE! The only commands implemented are the one used by the examples/google_cheese/ directory
  2. It's still experimental
  3. Not ready for Production
  4. Only 13 of the >90 WireProtocol Commands implemented
  5. I released it to get people interested in contributing. Not to provide a solution: it's not ready yet!

Requirements

  • PhantomJS version >= 1.6
  • Maven 3 (optional - only to easily build and run the Java-based examples)

Example (the only available so far :) )

There is plenty to do before this is usable, but if you can't wait to try PhantomJS's speed when it acts as a RemoteWebDriver Server, do the following:

1) Start GhostDriver on the terminal

$ phantomjs ghostdriver/main.js
Ghost Driver running on port 8080

2) Build and Launch the first Java-based example (Maven required):

$ cd ghostdriver/examples/google_cheese/
$ ./mvnexec-server-online.sh ghostdriver
